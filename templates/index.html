Intro

The purpose of this webapp is to provide a pipeline for all the basic steps involved in the lifecycle of a ML project including preprocessing, model training and hyperparamter optimization.

How to Use

1. Click on Start and upload your csv in the next page.
2. Select the problem statement - Regression problem or Classification problem.
3. If your csv is raw and not preprocessed at all then select yes in the preprocessing option else select no. If you select yes then minimal preprocessing techniques like Imputing, Standardization, Encoding will be applied to your data.
4. Click Submit.
5. Select the model which you want to use for training.

Important Note:

- Currently the web app only works with .csv files.
- Keep you target column at the very end in your csv file.
- Still working on this project so there maybe many errors left.
